# Pro Kabaddi App

## Overview
Pro Kabaddi App is a mobile application that provides real-time updates and news on the Pro Kabaddi League.

## Features
- Live score updates
- Player profiles
- News feed

## Technologies Used
- React Native
- JavaScript
- RESTful APIs
